## 🚀 Free for Anyone to Use (Open-Source Template)

I believe in building in public and sharing what I learn. **If you like the design and animations of this portfolio, feel free to use it!** 

This repository is completely open-source. You can fork it, customize the content, and use it to build your own personal portfolio. The design is tailored for developers, researchers, and students who want to present their work professionally.

### How to use this template:
1. **Fork** this repository to your GitHub account.
2. **Clone** it locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Mirror_Y.git
   ```
3. Customize the content in `src/components/` (Hero, About, Projects, etc.) to match your identity.
4. Deploy easily on [Vercel](https://vercel.com/) for free.

## ✨ Features

- **Immersive Visuals:** Seeded deterministic particle canvas, floating glowing elements, and responsive layout.
- **Framer Motion Animations:** Fluid page transitions, scroll-based section reveals, and an interactive custom cursor.
- **My Journey & Focus:** Sections dedicated to my HCI/AI projects, Japanese learning milestones, and professional timeline.
- **Project Showcase:** Glassmorphic cards highlighting real-world solutions.
- **Responsive & Premium Design:** Tailwind CSS engineered for pixel-perfect display across mobile, tablet, and desktop.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React 19)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## 💻 Running Locally

To run the project on your machine:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```
Navigate to [http://localhost:3000](http://localhost:3000) to see the application running.
